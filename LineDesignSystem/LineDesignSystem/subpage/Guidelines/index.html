UX GUIDELINES

UX GUIDELINES은 LINE 서비스 고유의 일관성있는 UX를 유지하기 위한 가이드라인입니다. 일반적 사용성 기준이 아닌 LINE 서비스의 특성과 LINE 메신저 사용 경험을 고려한 의미있는 UX 가이드라인을 제시합니다. 본 가이드라인은 각 프로젝트 담당자가 실질적으로 활용할 수 있도록 작성하였습니다. LINE에서 공통으로 사용되는 UX pattern을 정의하고, 상세한 use case 예시를 통해 각 화면 구성에 적합한 UX pattern의 type을 빠르게 선정할 수 있도록 도와줍니다.

  • Confirmation

    상황 별 구체적인 예시를 통해 confirmation 제공 방법에 대해 알 수 있습니다.
  • Selection

    Selection UX를 효과적으로 제공하기 위한 방법을 알 수 있습니다.
  • Modal

    일반적인 forward navigation과 modal의 차이점, modal의 종류와 사용 용도에 대해 알 수 있습니다.
  • New Feature Guide

    사용자가 앱 내 새로운 기능을 시작 할 수 있도록 도와주는 가이드를 제공합니다.
